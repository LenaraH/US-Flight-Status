# US-Flight-Status 2015

📌 Project Overview
This project shows flight activity in the United States in 2015. It explains how many flights were on time, delayed, or cancelled.
The dashboard gives an overview of flight performance across cities and airlines, helping viewers quickly see delay patterns and cancellation reasons.

🛠 Tech Stack
Power BI Desktop — used to build the dashboard
Power Query — used to clean and transform the data
DAX — used to create calculations and measures
Data Modeling — used to connect tables and build relationships
CSV Files — used as the data format

📂 Data Source
The data comes from a public dataset shown in an online tutorial video.
I downloaded the CSV files and used them to create the dashboard.

✨ Features
Flight totals showing how many flights were on time, delayed, or cancelled
City comparison to see which airports had the most flights
Airline delay rates to compare performance between airlines
Cancellation reasons displayed in a clear chart
Monthly trends

📎 Conclusion
This project shows how Power BI can be used to explore flight performance data in a clear and simple way. It helps users understand delays, cancellations, and trends across the United States in 2015.
